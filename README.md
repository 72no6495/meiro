# 迷路ジェネレーター

## プロジェクト概要
ユーザーが自由に設定を決めて迷路を生成し、その作成過程を視覚的に楽しむことができるWebアプリケーションです。作成した迷路は画像としてダウンロードでき、さらにゴールまでの道のりを赤い線で示した答えの画像もダウンロード可能です。

## 実装済みの機能
- **迷路の設定**: 幅（列数）、高さ（行数）、セルのサイズを自由に設定できます。
- **生成アニメーション**: 迷路が作られていく過程をアニメーションで視覚的に確認できます。生成速度の調整も可能です。
- **迷路の自動生成**: 深さ優先探索（Recursive Backtracker）アルゴリズムを用いてランダムな迷路を生成します。
- **経路探索**: 幅優先探索（BFS）を用いてスタートからゴールまでの最短経路を自動で計算します。
- **画像ダウンロード機能**:
  - 完成した迷路のみの画像（PNG）
  - スタートからゴールまでの経路（赤い線）を描画した答え付きの画像（PNG）

## 構成
- `index.html`: 全てのHTML、CSS（Tailwind CSSを利用）、JavaScript（Canvas API）が1つのファイルにまとめられています。

## 今後の開発の推奨ステップ
- **迷路の形状変更**: 四角形だけでなく、六角形や円形の迷路を生成する機能の追加。
- **カスタマイズ機能**: 壁の色、背景色、経路の線の色などをユーザーが自由に変更できる機能の実装。
- **難易度設定**: 経路の複雑さや分岐の数を調整するアルゴリズムの導入。
